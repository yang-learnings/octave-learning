# octave-learning

### Displaying
```octave
disp(sprintf('display 6 decimals: %0.6f', var));

disp(sprintf('display 3 decimals: %0.3f', var));
```
*formats*

```octave
format long;

format short;
```
### Matrices

```octave
a = [1 2; 5 6; 9 5]; # 3 by 2 matrix

v = 1:6; # -> [1 2 3 4 5 6]

v = 1:0.5:2; # -> [1 1.5 2]
```
